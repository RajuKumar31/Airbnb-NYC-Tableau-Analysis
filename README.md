# 🏙️ Airbnb NYC Market Analysis (Tableau)

## 📊 Project Overview

This project presents an interactive **Tableau analysis** of Airbnb listings in New York City.
It explores pricing trends, neighborhood distribution, room types, and customer engagement.

The project includes both a **Dashboard** for exploration and a **Story** for guided insights.

## 📌 Key Insights

* Manhattan commands the highest average listing prices
* Entire homes dominate both pricing and availability
* Lower-priced listings generate higher customer engagement (reviews)
* Premium listings show lower review activity, indicating niche demand

---

## 📖 Story Breakdown

The Tableau Story walks through the analysis step-by-step:

1. **Overview** – NYC Airbnb market summary
2. **Pricing by Room Type** – Average price comparison
3. **Neighborhood Analysis** – Borough-level pricing trends
4. **Price vs Reviews** – Engagement vs pricing relationship
5. **Key Insights** – Final conclusions
