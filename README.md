# Airbnb NYC Market Analysis (Tableau Project)

## 📊 Project Overview

This project analyzes Airbnb listings in New York City using Tableau.
It focuses on pricing trends, room types, neighborhood distribution, and customer engagement.

## 🔗 Live Dashboard

👉 https://public.tableau.com/app/profile/raju.kumar.s1115/viz/AirbnbNYCMarketAnalysisDashboard/AirbnbNYCPriceListingAnalysis

## 📌 Key Insights

* Manhattan has the highest average listing price
* Entire homes dominate both pricing and availability
* Lower-priced listings receive higher customer engagement (reviews)
* Premium listings show lower review activity
